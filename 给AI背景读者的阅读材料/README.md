# 给 AI 背景读者的阅读材料

这个目录面向已经熟悉机器学习、深度学习或 LLM，但只具备少量范畴论背景的读者。

阅读目标不是把“topos in AI”当成成熟算法，而是先学会判断一篇文章到底在使用哪一层数学结构：

1. 范畴、函子、自然变换、极限、伴随、Yoneda；
2. 预层、层、Grothendieck topology、site、层化；
3. Grothendieck topos、内部逻辑、stack；
4. 神经网络、Transformer 或 LLM 被编码进这些结构时，到底保留了什么性质。

## 目录

- [01-讲义](01-讲义): 教科书式讲义。适合从 AI 直觉出发，逐步补范畴论和 topos 语言。
- [02-背景综述](02-背景综述): 背景综述草稿。适合快速了解已有文献地图，但不应代替讲义学习。

## 推荐顺序

1. 先读 [01-讲义/topos_ai_neural_structures_lecture_zh.pdf](01-讲义/topos_ai_neural_structures_lecture_zh.pdf)。
2. 遇到不熟悉的范畴论术语，回到仓库根目录的 `docs/topos-ml-study-plan.md`。
3. 再读 [02-背景综述/topos_ai_neural_structures_zh.md](02-背景综述/topos_ai_neural_structures_zh.md)，把它当作文献索引，而不是定理论证。

# 给范畴论背景读者的阅读材料

这个目录面向熟悉范畴论、层、topos 或相关数学语言，但不熟悉现代深度学习和 LLM 算法细节的读者。

阅读目标是补齐 AI 侧的基本对象和统一符号，使后续讨论“神经网络/Transformer/LLM 是否能放入某个范畴论或 topos 框架”时，不会把算法对象理解成黑箱。

## 目录

- [01-AI算法讲义](01-AI算法讲义): 从监督学习、神经网络层、反向传播、CNN/GNN 到 Transformer 和 LLM 推理的讲义。

## 推荐顺序

1. 先读 [01-AI算法讲义/ai_algorithms_for_category_readers_zh.pdf](01-AI算法讲义/ai_algorithms_for_category_readers_zh.pdf)。
2. 重点掌握统一符号：数据集、参数、损失函数、层复合、梯度、attention、token 条件概率。
3. 再回到“给 AI 背景读者”的 topos 讲义，比较两边符号如何对接。

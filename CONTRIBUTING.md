# Contributing

Thanks for helping improve this learning map.

The main rule is precision: do not label broad category theory, topology, or AI
work as "topos theory" unless the connection is explicit.

## How To Add A Resource

Use this format:

```markdown
### Resource Title

- Link: <https://example.org>
- Authors: Name, Name
- Labels: Core Topos, AI-Relevant
- Level: beginner | intermediate | advanced
- Why it matters: one or two precise sentences.
- Caution: optional note about limitations or scope.
```

## Label Rules

- Use **Core Topos** only for real topos theory.
- Use **Sheaf-Theoretic** when the resource is about sheaves but not full topoi.
- Use **Applied Category Theory** for nearby categorical AI topics.
- Use **AI-Relevant** only when the AI connection is explicit.
- Use **Speculative Bridge** for promising but undeveloped connections.
- Use **Verify First** when the source is uncertain.

## Quality Bar

A good entry should:

- have a stable source URL;
- identify authors or maintainers;
- state the mathematical topic accurately;
- explain why a learner should care;
- avoid hype.

## Pull Request Checklist

Before opening a PR:

- check that links work;
- check that labels are justified;
- put speculative items in the right section;
- do not add generated-looking citations without verification;
- prefer fewer, better resources over long uncurated lists.
